## 📅 Progress

### Week 1–2 · HTML & CSS Fundamentals
- HTML structure: `<html>`, `<head>`, `<body>`
- Elements: buttons, links, paragraphs, inputs
- Attributes: `href`, `target="_blank"`
- CSS basics: colors, padding, border-radius
- Button design: hover, active states, box-shadow, transitions
- Transform: `translateY`, `scale` for click feedback
- Typography: font-family, HTML entities, `<span>`
- Developer Tools (F12): inspecting and reverse-engineering layouts
- Search bar design: background-image icon, margin: 0 auto

### Week 3 · CSS Layout
- `display: block` vs `inline-block` — understanding the difference
- Flexbox: `display: flex`, `flex-direction`, `gap`, `align-items`
- Box shadow syntax: `rgba()` — no space before parenthesis
- Built exercises: login form, Google search clone, survey layout
